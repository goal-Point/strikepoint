# KNGO

**WIN BIG WITH SKILL**

KNGO is a skill-based football competitions platform where the closest entry wins. Test your football knowledge and prediction skills to win real prizes.

## Features

- **Skill-Based Competitions** - Enter live football competitions where accuracy wins
- **Real Prizes** - Win cash prizes and rewards based on your performance
- **Daily Spin Wheel** - Spin once every 24 hours to win discount codes, free entries, and bonus entries
- **Winners Gallery** - See real winners and their prizes

## Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/KNGO.git
cd KNGO
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

### Build for Production

```bash
npm run build
npm start
```

## How It Works

1. **Browse Competitions** - View live skill-based football competitions
2. **Enter Your Prediction** - Submit your entry for the competition
3. **Closest Wins** - The entry closest to the actual result wins the prize
4. **100% Skill** - No luck involved, pure skill-based gaming

## Rewards System

KNGO features a daily spin wheel where users can win:
- 5% OFF discount codes
- 10% OFF discount codes
- 15% OFF discount codes
- FREE ENTRY to competitions
- +2 BONUS ENTRIES
- Try again prompts

Each user can spin once every 24 hours.
